# Scoop Bucket

Personal [Scoop](https://scoop.sh/) bucket for CLI tools by [@glebovdev](https://github.com/glebovdev).

## Usage

```powershell
scoop bucket add glebovdev https://github.com/glebovdev/scoop-bucket
scoop install <app-name>
```

## Available Apps

| App | Description |
|-----|-------------|
| [somafm](https://github.com/glebovdev/somafm-cli) | Terminal-based music player for SomaFM radio stations |

## Adding this bucket

```powershell
scoop bucket add glebovdev https://github.com/glebovdev/scoop-bucket
```

## Installing an app

```powershell
scoop install glebovdev/<app-name>
```

## Updating

```powershell
scoop update
scoop update <app-name>
```
